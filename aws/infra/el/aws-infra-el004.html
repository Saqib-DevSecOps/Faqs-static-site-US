1. How long is content cached in the Edge Location? [AWS: Infrastructure: AWS Edge Locations]
    1. "By default, each file stays in an edge location for 24 hours before it expires. The minimum expiration time is 0 seconds; there isn't a maximum expiration time."
    2. "Optionally, you can configure your origin server to add headers to the files, to indicate how long the files should stay in the cache in CloudFront edge locations."

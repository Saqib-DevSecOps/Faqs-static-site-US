    1. You may consider Regional Edge Caches as providing a backup function for Edge Locations
    2. When content is cached at the Edge Location, a copy is also placed in the Regional Edge Cache
    3. There are fewer Regional Edge Caches than there are Edge Locations, however each Regional Edge Cache is much larger than each Edge Location. As such it can hold content for longer.

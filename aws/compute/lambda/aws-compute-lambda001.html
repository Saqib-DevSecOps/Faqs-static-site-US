1. What is AWS Lambda? [AWS: Compute: AWS Lambda]
    1. A Compute offering
    2. It fulfills a Function-as-a-Service (FaaS) role
    3. A Serverless offering
    4. Allows a user to code without needing to pre-configure the infrastructure
    5. AWS Lambda was generally available in April 2015
    6. Note:
      1. Azure Functions and Google Cloud Functions provide functionality generally similar to AWS Lambda
      2. Azure Functions (general availability in November 2016)
      3. Google Cloud Functions (general availability in July 2018)

1. Is Function as a Service (FaaS) the same as Serverless? [AWS: Compute: AWS Lambda, Serverless]
    1. No
    2. FaaS (Function as a Service) is more a subset of serverless
    3. "Serverless is focused on any service category, be it compute, storage, database, messaging, api gateways, etc. where configuration, management, and billing of servers are invisible to the end user."
    4. "FaaS, on the other hand, while perhaps the most central technology in serverless architectures, is focused on the event-driven computing paradigm wherein application code, or containers, only run in response to events or requests."

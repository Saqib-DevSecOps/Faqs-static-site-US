1. Can you modify the instance size of a Reserved Instance? [AWS: Compute: Pricing]
    1. Yes, you can modify the instance size of a Reserved Instance if the following requirements are met.
      1. The platform is Linux/UNIX.
      2. The new Instance size must be in the same Instance family. I.e. you cannot modify an Reserved Instance from t2 to t3.
    2. The original and new Reserved Instance must have the same instance size footprint.

1. What flexibility is possible with Reserved Instances? [AWS: Compute: Pricing]
    1. For an RI to be size-flexible:
      1. It must meet the regional benefit criteria.
      2. It must have a platform value of Linux/UNIX
      3. It must not be a G4 instance
      4. It must not have a dedicated tenancy
      5. It must not be tied to a specific Availability Zone
      6. Regional RI apply the discount to any AZ in the Region
    2. "Regional Reserved Instances are purchased for a Region and provide Availability Zone flexibility. The Reserved Instance discount applies to instance usage in any Availability Zone in that Region."
    3. "Regional Reserved Instances also provide instance size flexibility where the Reserved Instance discount applies to instance usage within the instance family, regardless of size."

1. What's Root in AWS Organizations and how is it different from the Management account? [AWS: Management: AWS Organizations, Hierarchy, Accounts]
    1. Root is the parent organizational unit (OU) for all accounts and other OUs in your organization.
    2. Think of Root as the top-level folder within AWS Organizations. It is the container of all other resources in the Organization.
    3. This is not the AWS Root Account user.
    4. The AWS Organizations Management Account is an AWS IAM User with administrative privileges.  It is the account that was used to create the AWS Organizations service itself.
    5. Note: This could also be an AWS Root Account, however this goes against best practice advice.

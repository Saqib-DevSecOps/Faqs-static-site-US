1. Can you attach more than one SCP to an AWS Organizations entity? [AWS: Management: AWS Organizations, Policy]
    1. Every AWS Organizations entity (root, OU, account) must have at least one SCP attached at all times.
    2. You can have up to five SCP attached to any AWS Organizations entity.
    3. Note: You cannot remove the last SCP from an entity.
    4. Note: There is a default SCP named FullAWSAccess, which as the name implies, allows access to every operation.

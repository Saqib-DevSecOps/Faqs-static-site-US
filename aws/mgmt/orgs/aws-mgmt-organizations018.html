1. Can I disable the Consolidated Billing feature in AWS Organizations? [AWS: Management: AWS Organizations, Billing]
    1. No
    2. AWS Organizations has two available feature sets: All features and Consolidated Billing features.
    3. With all features enabled, you can use the advanced account management features available in AWS Organizations, including Consolidated Billing
    4. With consolidated billing features, only this feature is enabled

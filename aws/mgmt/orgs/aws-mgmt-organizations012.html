1. Is an account root user required to setup AWS Organizations? [AWS: Management: AWS Organizations, Accounts]
    1. Account Root is not required
    2. You can use the Account Root User to setup AWS Organizations, however it is not required and more importantly, it is NOT recommended
    3. An IAM user with appropriate administrator privileges is recommended
    4. To create an organization in AWS Organizations with an IAM user, it must have at a minimum the following privileges:
      1. organizations:CreateOrganization
      2. iam:CreateServiceLinkedRole

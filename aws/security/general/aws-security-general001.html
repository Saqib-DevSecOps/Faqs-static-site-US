1. Can I run penetration tests on my AWS environment? E.g. Can I test the security of my infrastructure? [AWS: Security: General]
    1. I would start by surveying the security services that AWS makes available. Take a look at Amazon Inspector, Amazon GuardDuty, Amazon Macie, these and other such services can be leveraged to help secure your environment
    2. Next, see the AWS Customer Support Policy for Penetration Testing for things the customer can and cannot do, with respect to running a penetration test and other security assessments on their AWS assets.
    3. Also find out about what AWS calls Provable Security, which involves using tools and techniques to "detect entire classes of misconfigurations that could potentially expose vulnerable data."

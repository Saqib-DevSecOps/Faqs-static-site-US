1. What is AWS Directory Service? [AWS: Security: AWS Directory Service, Introduction]
    1. "AWS Directory Service for Microsoft Active Directory, also known as AWS Managed Microsoft Active Directory (AD), enables your directory-aware workloads and AWS resources to use managed Active Directory (AD) in AWS."
    2. Composed of three services:
      1. AWS Directory Service for Microsoft Active Directory
      2. AWS AD Connector
      3. AWS Simple AD
    3. See also: Other directory types pricing

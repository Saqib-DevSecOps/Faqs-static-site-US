1. What are the different access methods for AWS services? [AWS: General: AWS Accounts]
    1. AWS Management Console – Browser-based access to procedures the AWS Management Console to perform tasks
    2. AWS Command Line Interface – tool for accessing AWS services via a local command line
    3. AWS SDKs – for supported programming languages, AWS provides an SDK for you to access AWS services. SDKs simplify authentication, integrate easily with your development environment, and provide access to AWS services
    4. Services API – for programming language where an SDK is not available for, you can access the Service API to make API requests directly, e.g. the CloudFront API.
    5. AWS Toolkits – a range of SDKs and Toolkits

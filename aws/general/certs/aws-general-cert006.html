1. Now that I've passed the AWS Certified Cloud Practitioner exam, which courses and websites are appropriate for AWS solution architect associate certification? [AWS: General: Certifications]
    1. Some references for preparing for the AWS Certified Solutions Architect Associate (CSA-A).
      1. Always start with the AWS Certified Solutions Architect – Associate page
        1. The Exam Guide is important and helpful
        2. Scroll down to the Prepare for your exam section for a number of helpful resources
      2. Check out AWS PartnerCast. E.g. here I searched “certification”. There are a variety of live and on-demand sessions. Note that the PartnerCast session expire after a while.
      3. AWS SkillBuilder has relevant offerings, e.g. this AWS PartnerCast - AWS Solution Architect Associate Certification Journey – English. Note, you need to login using your PartnerCentral account.
      4. Lots, and lots of resources highlighted on this AWS Ramp-Up Guide: Architect.
      5. Exam Readiness: AWS Certified Solutions Architect - Associate (Webinar)
      6. Take a look at Reddit. E.g. this subreddit on AWS Certifications. There is a relevant question, Resources for AWS Certified Solutions Architect Associate with some additional non-AWS resources highlighted here.
      7. Additional prep and exam prep sources (Note: I haven’t personally reviewed all of the resources below)
        1. AWS Certification Official Practice Question Set (SAA-C02 - English)
          1. If you get a 403 FORBIDDEN Error with the above, click on the hamburger menu on top left (3 horizontal bars), click on Learner dashboard:
          2. Scroll down to Browse all courses, and search for “AWS Certification Official Practice Question Sets”:
          3. You will see a list of courses. Find the specific one you need, e.g. AWS Certified Solutions Architect - Associate, and the one in your preferred language.
            Click on it:
          4. Click on ENROLL then START LEARNING NOW.
            Click START LEARNING NOW again.
          5. Click START HERE, then read and agree to the code of conduct.
          6. Click See more to open the test, then click Start.
          7. They have 20-questions for a number of AWS exams including the CSA-A.
          8. Note: the access expires after 1-year of sign-up.
        2. There is also the Exam Prep: AWS Certified Solutions Architect - Associate:
        3. There are other options available with a subscription plan, e.g.
          1. AWS Certified Solutions Architect - Associate Official Practice Exam (SAA-C03 - English)
          2. Exam Prep: AWS Certified Solutions Architect - Associate (with Practice Material)
            It presents 65 questions in the format of the real exam, with a time limit. This practice exam aligns with the SAA-C03  version of the exam
        4. AWS Training and Certification has a number of Exam Readiness classes. This is one for the CSA-A.
          1. Find an available session that meets your schedule. You can filter by training provider.
          2. Note the above are 5-day ILT Exam Readiness Intensive Workshop, which have a cost.
        5. Get exam ready with the  Solutions Architect, SysOps Administrator, and Developer Resource Hubs

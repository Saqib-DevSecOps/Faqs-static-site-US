1. How did you prepare for your AWS CSA-A exam? [AWS: General: Certifications, CSAA]
    *Note: Not an exhaustive list. It will be different for everyone, based on skill-level, availability, etc.*
    1. I read, did hands-on labs, and practice exams
      1. Also, for me it helped to write things down. As such, I took a lot of notes as I learnt new things. Writing it down helps to solidify the information for me.
      2. As a bonus, I can use the material to teach others.
    2. If I remember correctly it took about 3 months
    3. Read:
      1. AWS Exam Guides
        1. There are exam guides for each of the AWS certifications
        2. Go to the exam site (e.g. AWS Certified Solutions Architect - Associate) and click on the Download the exam guide link.
      2. AWS Whitepapers, e.g.
        1. AWS Well-Architected Framework
        2. An Overview of the AWS Cloud Adoption Framework
        3. How AWS Pricing Works
      3. AWS Websites, e.g.
        1. AWS FAQs: https://aws.amazon.com/faqs/
        2. High-level overview of AWS services: https://aws.amazon.com
        3. AWS Products, by category, e.g. https://aws.amazon.com/products/databases/
        4. Deep dive technical documentation site: https://docs.aws.amazon.com
      4. Miscellaneous
        1. AWS content on twitch.tv, e.g. search AWS.
        2. AWS Power Hour: Architecting
        3. AWS Training and Certification
          1. Classroom Training
          2. e.g. Architecting on AWS
      5. Searching the web for discussions on forums
    4. Hands-on labs
      1. Used (at the time), the AW content at Qwiklabs. That is no longer available. Today you would leverage the AWS Builder Labs
        1. If you go this route, I recommend you pay for a subscription, and get access to the self-paced lab environment.
      2. Get access to an AWS Account
        1. If you have access to a development account at work you can use for this, do that.
        2. Or you can create your own AWS Account via the AWS Free Tier program.
        3. This is where you give AWS your email address and credit card number, and they give you access to the full-fledged AWS environment.
        4. As part of this program, you get some services available for free for, in some cases up to 12 months, from the time you sign up for the AWS Free Tier account.
    5. Practice exams
      1. Sample questions
        1. Go to the exam site (e.g. AWS Certified Solutions Architect - Associate) and click on the Download the sample questions link.
        2. Access Exam Prep information on AWS Skill Builder. You will find free and subscription-based content. Search for "exam prep":

1. What tasks require the credentials of the AWS Account Root User? [AWS: General: AWS Account, Permissions]
    1. Most tasks can be carried out in an AWS Account using an IAM User with appropriate privileges.
    2. There are very few tasks that require the Account Root User, and they include:
      1. Closing the account
      2. Configuring MFA Delete on an Amazon S3 bucket
      3. Changing or canceling AWS Support Plan
    3. Note: If the AWS Account is a Member Account in an AWS Organizations service, the Root User of the Member Account might be restricted using a Service Control Policy (SCP), from carrying out one or more of the tasks that only the Root User can do.

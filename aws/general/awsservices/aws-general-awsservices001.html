1. Which AWS Services are free? [AWS: General: AWS Services]
    1. We can opt to talk about a number of things here, including the AWS Free Tier account, which allows new AWS customers try out AWS services free of charge, up to a limit for each service.
    2. Outside of the AWS Free Tier, there are several free services categories:
      1. Threshold-base free services are services where the service, or some aspect of the service is made free, up to a certain threshold, beyond that, charges apply. For example, AWS Lambda provides 1 million requests free per month, indefinitely
      2. Free services that only charge for underlying services. For example, AWS Elastic Beanstalk is a service for developers to quickly deploy and manage applications in the AWS Cloud. There is no charge for using AWS Elastic Beanstalk. You pay for AWS resources (e.g. EC2 instances or S3 buckets) you create to store and run your application
    3. Find a lot of the free services by visiting the URL with this format: aws.amazon.com/free/<category-name>, where <category-names> include: analytics, compute, database, machine-learning, networking, storage, etc. e.g. https://aws.amazon.com/free/storage/.
      Get a complete list of category names from: https://aws.amazon.com/free.

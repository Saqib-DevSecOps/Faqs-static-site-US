1. Can I run Windows Containers on a Linux host? [AWS: Container: Technology]
    1. TBD
    2. Short answer, No, though it's technically possible.
    3. You can run Linux Containers on a Windows host because the Windows host creates a Linux VM on the host and runs the Linux Container on the host.
    4. "Docker for Windows allows you to simulate running Linux containers on Windows, but under the hood a Linux VM is created, so still Linux containers are running on Linux, and Windows containers are running on Windows."
    5. It can do the same (in reverse) to run Windows Containers on a Linux host. However, it hasn't yet, perhaps due to License restrictions and management overhead.
    6. References:
      1. Virtualization via Containers (cmu.edu)
      2. The Differences Between Linux and Windows Containers (containerjournal.com)

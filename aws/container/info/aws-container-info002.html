1. Does ECS use Docker Swarm? [AWS: Container: Technology]
    1. ECS is a container orchestration solution
    2. Self-service alternatives to ECS include Docker Swarm (swarm-mode), and Kubernetes
    3. Swarm is Docker’s own orchestration tool and makes use of the standard Docker API and networking
    4. Kubernetes is an open source container orchestration tool, designed to support large enterprise architectures
    5. There is no confirmation that ECS uses Docker Swarm. It likely uses an AWS proprietary solution for orchestration in ECS, and at most, compatible with Swarm

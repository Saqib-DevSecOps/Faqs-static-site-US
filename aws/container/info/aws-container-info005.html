1. What is AWS Fargate? [AWS: Container: Technology]
    1. "AWS Fargate is a compute engine for Amazon ECS/EKS that runs containers without requiring you to deploy or manage the underlying Amazon EC2 instances.
    2. With Fargate, you specify an image to deploy and the amount of CPU, a proportional amount of memory it requires.
    3. You are charged based on the vCPU, memory, storage and duration your tasks consume
    4. Fargate handles the updating and securing of the underlying Linux OS, Docker daemon, and ECS agent as well as all the infrastructure capacity management and scaling."

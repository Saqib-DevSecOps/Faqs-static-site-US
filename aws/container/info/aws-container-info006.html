1. What is an Amazon EC2 Instance?
    2. [AWS: Compute: Instances]
    1. Amazon EC2 (Elastic Compute Cloud) provides secure and resizable compute capacity for virtually any workload.
    2. "Amazon EC2 Instances allow businesses to benefit from powerful and expensive hardware, without the cost of purchasing, configuring, and maintaining that hardware.
    3. There’s essentially no limit to the number of virtual machines and servers you can rent, which means you can scale up and down rapidly to respond to peaks and troughs in demand, develop applications faster, and get much better value for money and flexibility than is possible from on-premises hardware."
    4. "An EC2 Instance is a virtual machine that runs on a remote host machine (“hypervisor”) in a similar way to how a virtualized operating system runs on a desktop computer or server.
    5. The instance type is determined by the configuration of the hypervisorâ€”which can be “General Purpose,” “Compute Optimized,” “Memory Optimized,” etc. Consequently, the hardware used will dictate the memory, storage, compute capabilities, and performance of the virtual machine."

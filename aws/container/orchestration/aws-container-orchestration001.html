1. Will AWS continue to offer both ECS and EKS longer term - they seem very similar? [AWS: Container: Orchestration]
    1. Both services will continue to be supported, as far as I can tell
    2. They are both AWS Container Orchestration/Management solutions, however general difference between ECS and EKS is:
      1. ECS supports Docker as the container engine; EKS supports Docker and other container engines
      2. ECS orchestrates only Docker Container Engines; EKS orchestrations a variety of Container Engines, including Docker
      3. ECS is tightly integrated with other AWS services; EKS is integrated with other AWS services, and it is open-source Kubernetes conformant
      4. ECS offers simplicity, making it easier to deploy standard architectures; EKS offers ability to deploy a more enterprise architecture

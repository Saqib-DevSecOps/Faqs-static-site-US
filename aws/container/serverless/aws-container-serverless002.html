1. What are the Serverless offerings in AWS? [AWS: Container: Serverless]
    1. "AWS offers technologies for running code, managing data, and integrating applications, all without managing servers."
    2. A Serverless service is one where no input is needed from the subscriber on what the server infrastructure of the service should look like.
    3. The following are the serverless offerings in AWS, by category:
    4. Compute
      1. AWS Lambda
      2. AWS Fargate
    5. Datastores
      1. Amazon Aurora Serverless
      2. Amazon DynamoDB
      3. Amazon S3
      4. Amazon RDS Proxy
    6. Integration
      1. AWS SNS
      2. AWS SQS
      3. AWS Kinesis Data Firehouse
      4. AWS Step Functions
      5. Amazon EventBridge
      6. AWS API Gateway
      7. AWS AppSync
